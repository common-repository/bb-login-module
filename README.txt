== Description ==

Add an easy to use Beaver Builder Login/Logout button module. Works for single sites and multisite.

== Prerequisite: ==

- Beaver Builder

== Features: ==

- This addon provides Login functionality for your WP Single/Multi Site users
- The users can login/logout to their sites without going to WP Dashboard
- Admin can drag and drop the login Modal from Beaver Builder to any page
- Admin can edit the login Modals Button text, Style and Background
- Admin can remove the login Modal by simply deleting it from the page
- Works with both SingleSite and MultiSite
- Works for all user roles

== Installation ==

#### Minimum System Requirements

Beaver Builder Login Module For Beaver Builder Requires

+ Beaver Builder
+ PHP 7.2 or later
+ MySQL 5.6 or later
+ WordPress 4.0 or later
Before installation please make sure you have latest Beaver Builder installed.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
